# [+] Struktur Program C++ [+]

```
1. Statements : Intruksi yang diberikan secara baris per baris untuk dijalankan oleh mesin

2. Variables : Lokasi penyimpanan yang digunakan untuk menampung sebuah data atau informasi. Contoh, lokasi1, variabel2

3. Literals : Simbol-simbol yang digunakan untuk mengisi suatu variabel. Seperti angka 5, 10, atau 'kalimat atau string ini'

4. Operators : Simbol-simbol yang digunakan untuk mengubah nilai dari suatu variabel dengan melibatkan satu atau lebih variabel dan literal. Contoh, tanda (+) yang dapat menambahkan isi dari bilangan1 dan bilangan2
```
<b>Beberapa operator pada C++</b>
```
1. Operator - yang berfungsi sebagai operator pengurangan
1. Operator * yang berfungsi sebagai operator perkalian
1. Operator / yang berfungsi sebagai operator pembagian
```
