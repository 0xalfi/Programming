# [+] Tipe Data Dasar [+]

```
1. Null Type : Tipe data null digunakan untuk menyimpan nilai kosong atau tidak ada yang dinyatakan dengan NULL

2. Boolean Type : Tipe data boolean atau bool digunakan untuk menyimpan nilai kebenaran yaitu true atau false dari suatu ekspresi logika

3. Numeric Type : Tipe data yang digunakan untuk menyimpan data berupa angka. Terdapat dua macam tipe data numeric,
- int : untuk menyimpan bilangan bulat (3,2,1,0,-1,-2,-3,...) dan
- float : untuk menyimpan bilangan riil (0.5, 4.3, 2.35)

4. Text Type : Tipe data string atau str digunakan untuk menyimpan data teks. Tipe data string diapit dengan tanda kutip (" ")

5. Character : Tipe data yang digunakan untuk menyimpan data berupa karakter. Tipe data karakter diapit dengan tanda kutip tunggal ('')
```
