# [+] Operator [+]

```
1. Arithmetic Operators : + - * / %
2. Assignment Operators : += -= *= /= %=
3. Comparison Operators : == != > < >= <=
4. Logical Operators : && || not
```

# [+] Nilai Prioritas Operator [+]

```
1. Operator () memiliki nilai prioritas 7 dan digunakan untuk grouping
2. Operator %,*,/ memiliki nilai prioritas 6 dan digunakan untuk modulus, perkalian,, dan pembagian
3. Operator +,- memiliki nilai prioritas 5 dan digunakan untuk penambahan, pengurangan
4. Operator <,>,<=,>=,==,!= memiliki nilai prioritas 4 dan digunakan untuk comparison
5. Operator not() memiliki nilai prioritas 3 dan digunakan untuk operator logika negasi
6. Operator && memiliki nilai prioritas 2 dan digunakan untuk operator logika konjungsi (and)
7. Operator || memiliki nilai prioritas 1 dan digunakan untuk operator logika disjungsi (or)
```
